# よくある質問（FAQ）

## 📌 基本的な質問

### Q1: この提案の一番のポイントは何ですか？

**A**: 既存のテンプレート編集に**音声入力**を加えることです。

「背景を青にして」「もっと大きく」などと話すだけで、直感的にデザインをカスタマイズできます。スマホでも操作しやすく、**業界初**の試みになります。

---

### Q2: なぜ音声入力が必要なのですか？

**A**: 3つの理由があります：

1. **操作が簡単**: 特にモバイルで細かいボタン操作が不要
2. **時間短縮**: 5分 → 3分に短縮可能
3. **差別化**: 競合にない独自機能

実際、ユーザーは「ここをこうしたい」という直感的な言葉で指示したいニーズがあります。

---

### Q3: 音声認識の精度は大丈夫ですか？

**A**: 対策を用意しています：

- **認識結果を必ず確認画面で表示**
- **従来のテキスト編集も併用可能**
- **よく使う指示は学習して精度向上**

最初は60〜70%の認識率でも、ユーザーは「便利だな」と感じてもらえる設計にします。

---

### Q4: 3Dプレビューは本当に必要ですか？

**A**: 「思ったより小さかった/大きかった」というクレームを防ぐためです。

**メリット**:
- 注文前にサイズ感を確認できる
- 返品・修正依頼が減る
- 顧客満足度が上がる

シンプルな3D表示なら技術的にも実現しやすいです。

---

### Q5: AIアドバイスは過剰ではないですか？

**A**: オプション機能として控えめに表示します。

```
💡 ヒント: この用途なら黄色と黒がおすすめです
   [適用する] [無視する]
```

押し付けがましくなく、「へぇ、そうなんだ」と思ってもらえる程度の提案です。

### Q7: 運用コスト月¥5,000は本当に収まりますか？

**A**: はい、以下の工夫で実現します：

**内訳**:
- サーバー費用: ¥1,000〜2,000/月
  - Vercel（フロント）: 無料枠
  - Firebase（バック）: 無料〜¥1,000/月
- AI API使用料: ¥2,000〜3,000/月
  - 音声認識: Web Speech API（無料）
  - GPT-4: 月100回程度に制限

**コスト削減の秘訣**:
1. **音声認識は無料API優先**: Web Speech APIをメインに使用
2. **AIアドバイスは事前生成**: よくある組み合わせは保存して再利用
3. **キャッシング活用**: 同じ質問には保存済み回答を返す
4. **3Dモデルは静的配信**: CDN経由で追加コストなし

初期ユーザー数が少ない間は、ほぼ無料で運用可能です。

### Q8: 投資回収はいつ頃ですか？

**A**: シミュレーション例：

```
前提: 月間200注文、平均単価$150、手数料15%

月間収益: 200 × $150 × 15% = $4,500
運用コスト: -¥5,000（約$35）
純利益: 約$4,465/月

開発費$50,000 ÷ $4,465 = 約11ヶ月で回収
```

注文数が増えれば、より早く回収できます。

**A**: 実績のある技術のみ使用します：
