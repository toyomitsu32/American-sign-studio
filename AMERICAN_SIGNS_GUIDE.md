# アメリカ標識デザインガイド

## 🇺🇸 なぜ「アメリカ標識専門」なのか

### アメリカ標識の特徴

#### 1. 独特のデザイン文法
- **Highway Gothic フォント**: 1948年から使われる道路標識標準
- **Clearview フォント**: 2000年代の新世代フォント
- **色の意味**: 赤=禁止、黄=警告、青=情報、緑=案内

#### 2. MUTCD規格
**Manual on Uniform Traffic Control Devices**

アメリカ全土で統一された標識規格：
- サイズ基準
- 色の使用ルール
- 反射材の要件
- 設置高さ・距離

#### 3. 文化的背景
- **ルート66**: ビンテージアメリカン標識の象徴
- **ダイナー文化**: ネオン風・レトロスタイル
- **インダストリアル**: 工場・倉庫の実用的スタイル

## 📏 標識サイズガイド

### 用途別推奨サイズ

| 用途 | 推奨サイズ（インチ） | 設置場所 |
|------|-------------------|---------|
| 駐車場標識 | 18" × 24" | ガレージ、駐車スペース |
| 店舗営業時間 | 12" × 18" | 店舗入口 |
| 安全警告 | 24" × 36" | 倉庫、工場 |
| 案内標識 | 12" × 12" | オフィス内 |
| 大型道路標識 | 48" × 48" | 幹線道路 |

### 視認距離の目安

```
文字サイズ 3" → 30フィート（約9m）から読める
文字サイズ 6" → 60フィート（約18m）から読める
文字サイズ 12" → 120フィート（約36m）から読める
```

## 🎨 色の使用ルール（MUTCD準拠）

### 公式カラーコード

| 色 | 意味 | 使用例 | HEX |
|----|------|--------|-----|
| 🔴 **Red** | 禁止・停止 | STOP, NO PARKING | #DA291C |
| 🟡 **Yellow** | 警告・注意 | CAUTION, WARNING | #FFD100 |
| 🟢 **Green** | 案内・許可 | 方向案内、GO | #00843D |
| 🔵 **Blue** | 情報・サービス | 駐車場、案内 | #003F87 |
| ⚫ **Black** | テキスト・コントラスト | 文字、図形 | #000000 |
| ⚪ **White** | 背景・規制 | 背景、枠線 | #FFFFFF |
| 🟠 **Orange** | 工事・一時 | 工事中、迂回 | #FF6600 |

### 組み合わせルール

**最も視認性が高い組み合わせ**:
1. イエロー背景 × ブラック文字（警告系）
2. ブルー背景 × ホワイト文字（情報系）
3. ホワイト背景 × レッド文字（禁止系）

## ✍️ フォントガイド

### Highway Gothic シリーズ

アメリカ道路標識の標準フォント（1948年〜）

```
Series B: 細字 - 小さい標識用
Series C: 中字 - 一般的な標識
Series D: 太字 - 大型標識
Series E: 極太 - 高速道路標識
Series F: 超極太 - 特大標識
```

**特徴**:
- 高速走行中でも読みやすい
- 角が丸く視認性◎
- 大文字専用

### Clearview フォント

新世代標識フォント（2004年〜）

**特徴**:
- Highway Gothicより夜間視認性15%向上
- より現代的なデザイン
- 小文字にも対応

### レトロ・ビンテージフォント

**人気フォント**:
- **Futura Bold**: モダンビンテージ
- **Impact**: 太字・インパクト大
- **Cooper Black**: 丸みのあるレトロ

## 🏪 用途別デザイン例

### 1. 駐車場標識（Private Parking）

```
┌──────────────────────┐
│      NO PARKING      │
│                      │
│  🚫 Violators will  │
│     be towed        │
│                      │
│   At Owner's Expense │
└──────────────────────┘

推奨:
• サイズ: 18" × 24"
• 色: イエロー背景 × ブラック文字
• フォント: Highway Gothic Series D
• 素材: アルミニウム（反射コーティング）
```

### 2. 店舗営業標識（Business Hours）

```
┌──────────────────────┐
│       OPEN           │
│                      │
│   Monday - Friday    │
│     9:00 - 17:00     │
│                      │
│     Saturday         │
│    10:00 - 15:00     │
│                      │
│   Sunday: CLOSED     │
└──────────────────────┘

推奨:
• サイズ: 12" × 18"
• 色: ホワイト背景 × ブラック文字
• アクセント: グリーン（OPEN）、レッド（CLOSED）
• フォント: Clearview
```

### 3. 安全警告標識（Caution）

```
┌──────────────────────┐
│     ⚠️ CAUTION       │
│                      │
│   WATCH YOUR STEP    │
│                      │
│   Uneven Surface     │
└──────────────────────┘

推奨:
• サイズ: 18" × 24" 以上
• 色: イエロー背景 × ブラック文字
• アイコン: ⚠️ 三角警告マーク（MUTCD規格）
• フォント: Highway Gothic Series E（太字）
```

### 4. ビンテージ・ルート66スタイル

```
┌──────────────────────┐
│    🏍️ ROUTE 66       │
│                      │
│   GARAGE & DINER     │
│                      │
│  Est. 1952           │
│                      │
│  "The Main Street    │
│   of America"        │
└──────────────────────┘

推奨:
• サイズ: 24" × 30"
• 色: レッド・ホワイト・ブルー
• エフェクト: ヴィンテージ加工（錆・色褪せ）
• フォント: Futura Bold
```

## 🏗️ 素材ガイド

### アルミニウム（最も一般的）

**特徴**:
- 軽量で耐久性◎
- 反射コーティング可能
- 屋外使用10年以上

**価格帯**: $50〜$150

**用途**: 駐車場、屋外標識

---

### プラスチック（コスト重視）

**特徴**:
- 軽量・安価
- 屋内向け
- 耐久性△

**価格帯**: $20〜$60

**用途**: 店内案内、一時的な標識

---

### 木製（ビンテージ風）

**特徴**:
- ナチュラル・温かみ
- カスタム加工しやすい
- 屋外は要メンテナンス

**価格帯**: $80〜$200

**用途**: カフェ、ガレージ、ビンテージ店舗

---

### スチール（インダストリアル）

**特徴**:
- 重厚感・耐久性最強
- 重い（設置工事必要）
- 錆加工でビンテージ風

**価格帯**: $100〜$300

**用途**: 工場、倉庫、大型施設

---

## 📍 State（州）別の特徴

### カリフォルニア（CA）

**スタイル**: カジュアル・ビーチ風
- 明るい色（オーシャンブルー、サンイエロー）
- サーフ・パームツリーモチーフ
- リラックスした雰囲気

### テキサス（TX）

**スタイル**: ビッグ&ボールド
- 大きめサイズ
- テキサス州旗カラー（レッド・ホワイト・ブルー）
- カウボーイ・ウエスタンモチーフ

### ニューヨーク（NY）

**スタイル**: アーバン・コンパクト
- コンパクトなサイズ
- モノクローム・ミニマル
- モダンでスタイリッシュ

### フロリダ（FL）

**スタイル**: トロピカル・リゾート
- 明るい色（ターコイズ、コーラル）
- ヤシの木・サンセットモチーフ
- バケーション感
