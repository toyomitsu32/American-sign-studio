# American Sign Studio - AI自然言語デザインサービス

> 「こんな標識が欲しい」を話すだけで、本場アメリカンスタイルの標識デザインが完成

## 💡 提案の背景

従来のテンプレート編集では「テキストを変える」「色を選ぶ」だけで、**本当のアメリカンデザイン**は生まれません。

そこで、ユーザーが自然な言葉で要望を伝えると、AIが**アメリカ標識の文化・デザイン文法・MUTCD規格**を理解してプロフェッショナルなデザインを提案するサービスを作りたいと考えています。

## 🇺🇸 AIが理解すること

### 1. アメリカ標識の文化
- **ルート66**: ビンテージ・レトロの象徴
- **Highway Gothic**: 1948年からの標準フォント
- **ダイナー文化**: ネオン・クラシックカー
- **インダストリアル**: 工場・倉庫の実用美

### 2. MUTCD規格（米国交通標識規格）
- 色の使用ルール（赤=禁止、黄=警告、青=案内）
- サイズ基準
- 視認距離の計算
- フォント・レイアウト規定

### 3. デザイン文法
- **駐車場標識**: イエロー×ブラック、角張ったフォント
- **ビンテージ**: 錆加工、レトロフォント、星条旗カラー
- **モダン**: ミニマル、サンセリフ、モノクローム
- **ウエスタン**: カウボーイ、ステンシル風、ブラウン系

### 4. State（州）別の特徴
- **カリフォルニア**: ビーチ・サーフカルチャー
- **テキサス**: ビッグ&ボールド・カウボーイ
- **ニューヨーク**: アーバン・コンパクト

## 💬 自然言語入力の例

### 例1: 駐車場標識

**ユーザー入力**:
```
「うちの駐車場用。無断駐車されて困ってるから、
 ちゃんと怖い感じで。レッカー移動するって書いて」
```

**AI生成デザイン**:
```
┌─────────────────────────────┐
│    ⚠️ NO PARKING           │
│   PRIVATE PROPERTY          │
│                             │
│  Unauthorized vehicles      │
│   will be TOWED             │
│  at owner's expense         │
│                             │
│   Violators will be         │
│   prosecuted                │
└─────────────────────────────┘

デザイン仕様:
• サイズ: 18" × 24"
• カラー: イエロー背景 × ブラック文字
• フォント: Highway Gothic Series E（太字）
• アイコン: 警告三角マーク（MUTCD準拠）
• 素材: 反射アルミニウム推奨
• トーン: 威圧的・明確

理由:
✓ 「怖い感じ」→ 警告色＋強い言葉遣い
✓ 「レッカー移動」→ MUTCD標準文言を使用
✓ アメリカの駐車場標識の定番スタイル
```

### 例2: ビンテージガレージ

**ユーザー入力**:
```
「クラシックカーのガレージに飾りたい。
 50年代のアメリカンな感じ。
 ルート66みたいなやつ」

**AI生成デザイン**:
```

│                             │

│   Since 1952                │
│                             │

  - セピアトーン

理由:
✓ 「50年代」→ Futura Bold（当時の人気フォント）

✓ ヴィンテージ加工で本物感

- 軍用・インダストリアル

- DIY感
