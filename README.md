# AI標識デザインサービス - MVP提案書

> 「こんなサービスどうでしょう？」

## 💡 提案の背景

現在の標識制作サービスでは、ユーザーがテンプレートを選んでテキストを変更する基本的な機能はありますが、次のステップとして**AI技術を活用したより直感的なデザイン体験**を提供できないかと考えています。

## 🎯 提案内容：3つの新機能

### 1. 🎤 AI音声カスタマイズ（メイン機能）

**既存**: テキストボックスに入力して色を選択
**提案**: 音声で自然に指示

```
ユーザー: "背景を青にして、テキストはもっと大きく"
システム: → 即座に反映
```

**メリット**:
- 操作がシンプル（特にモバイル）
- 直感的なデザイン変更
- **差別化ポイント**になる

---

### 2. 🖼️ 簡易3Dプレビュー

**既存**: 平面の画像プレビュー
**提案**: 簡単な3D表示で実際のサイズ感を確認

- 回転して確認できる
- 壁に設置したイメージを表示
- 「これくらいの大きさ」が分かりやすい

**メリット**:
- 注文後のギャップが減る
- 顧客満足度アップ
- 返品・クレーム削減

---

### 3. 💬 AIデザインアドバイス

**提案**: AIが用途に応じた提案をする

```
例：駐車場標識を選択
AI: 「この用途なら黄色と黒の組み合わせが視認性高いですよ」
AI: 「テキストは24pt以上がおすすめです」
```

**メリット**:
- デザイン知識がなくても安心
- プロフェッショナルな仕上がり
- 規格準拠をサポート

## 📱 シンプルなユーザー体験

```
ステップ1: テンプレート選択（今まで通り）
   ↓
ステップ2: 🎤 音声で「青にして」「大きくして」
   ↓
ステップ3: 3Dでくるっと回して確認
   ↓
ステップ4: 注文（今まで通り）
```

**所要時間**: 既存5分 → **新3分**に短縮

## 💰 概算コストと期間（MVP版）

### 開発期間: **6〜8週間**

| フェーズ | 期間 | 内容 |
|---------|------|------|
| Phase 1 | 2週間 | 音声入力の基本実装 |
| Phase 2 | 2週間 | 簡易3Dプレビュー |
| Phase 3 | 2週間 | AIアドバイス機能 |
| テスト | 1〜2週間 | バグ修正・調整 |

### 概算コスト: **$40,000〜$60,000**

- 開発者2名 × 8週間
- AI API利用料（初期）
- インフラ費用

### 運用コスト: **月¥5,000以内（約$35）**

**内訳**:
- サーバー費用: ¥1,000〜2,000/月（Vercel/Firebase無料枠+有料少々）
- AI API使用料: ¥2,000〜3,000/月
  - 音声認識: Web Speech API（無料）をメインに使用
  - GPT-4は月100回程度の使用に制限
  - キャッシング活用で再利用

**コスト削減の工夫**:
- 音声認識はブラウザ標準API（無料）を優先
- AIアドバイスは事前生成して保存（毎回API呼ばない）
- 3Dモデルは静的ファイルで配信（追加コストなし）
- 初期は無料プランの範囲内で運用可能

### 収益への影響（予想）

- 音声機能による**注文完了率 +5〜10%**
- 3Dプレビューによる**返品率 -20〜30%**
- 「AI対応」による**新規顧客 +15〜20%**

## 🔧 技術的な実現性

### 音声入力
- **Web Speech API**（無料、ブラウザ標準）
- フォールバック: Google Speech-to-Text
- 実装難易度: ⭐⭐☆☆☆（低）

### 3Dプレビュー
- **Three.js**（無料、実績多数）
- シンプルな立方体モデル
- 実装難易度: ⭐⭐⭐☆☆（中）

### AIアドバイス
- **OpenAI GPT-4 API**
- テンプレート毎のルール設定
- 実装難易度: ⭐⭐☆☆☆（低）

**→ 既存技術の組み合わせで実現可能**

## 📊 想定されるメリット・デメリット

### ✅ メリット

1. **競合との差別化**
   - AI音声対応は業界初クラス
   - 「最新技術」としてPR可能

2. **ユーザー体験向上**
   - 操作が簡単・楽しい
   - 完成イメージが明確

3. **業務効率化**
   - 問い合わせ減少（プレビューで確認済み）
   - 返品・修正依頼の削減

### ⚠️ デメリット・リスク

1. **音声認識の精度**
   - 対策: テキスト編集も併用可能に
   - 認識結果を必ず確認画面で表示

2. **AI APIコスト**
   - 対策: キャッシュ活用で再利用
   - 事前生成して保存（毎回APIを呼ばない）
   - 月¥5,000以内に収まるよう設計

3. **ユーザーの慣れ**
   - 対策: チュートリアル動画
   - 従来の編集方法も残す
