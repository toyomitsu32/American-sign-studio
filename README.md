# SignCraft AI - AI搭載標識デザインサービス

> ユーザーが音声とテキストで簡単にカスタマイズできる、次世代のアメリカ標識制作プラットフォーム

## 📋 プロジェクト概要

SignCraft AIは、最新のAI技術を活用してユーザーが直感的に標識をデザインできるWebサービスです。テンプレートベースのシンプルな操作に、音声入力とAIによるデザイン提案を組み合わせることで、誰でも簡単にプロフェッショナルな標識を制作できます。

### 🎯 コアコンセプト

1. **シンプル**: テンプレート選択 → カスタマイズ → 注文（5分で完了）
2. **AI活用**: 音声入力でデザイン変更、AIによる最適化提案
3. **リアルタイム**: 3Dプレビューで完成イメージを即座に確認
4. **シームレス**: 制作会社への直接発注で迅速な納品

## 🌟 主要機能

### ユーザー向け機能

#### 1. 直感的なテンプレート選択
- 6つの主要カテゴリー
  - 🚫 安全・警告標識
  - 🚗 駐車場・交通標識
  - 🏢 ビジネス・店舗標識
  - ℹ️ 案内・情報標識
  - 🏠 プライベート・住宅用
  - 🎨 カスタムデザイン

#### 2. AI音声入力カスタマイズ
```
ユーザー: "タイトルをWelcomeに変更して背景を青にして"
システム: 即座にデザインを反映
```

#### 3. リアルタイム3Dプレビュー
- 360度回転可能な3Dモデル
- 実際の設置環境での合成プレビュー
- サイズ感の視覚化

#### 4. 自動見積もり・注文
- サイズ・素材・数量に応じた即時見積もり
- 製作期間の自動計算
- ワンクリック注文

### 制作会社向け機能

- 注文管理ダッシュボード
- デザインデータの自動受信
- ステータス管理・追跡
- 顧客コミュニケーション機能

## 🏗️ 技術スタック

### フロントエンド
- **Framework**: React 18 + TypeScript
- **Styling**: Tailwind CSS
- **3D Graphics**: Three.js / React Three Fiber
- **State Management**: Zustand
- **Voice Input**: Web Speech API + Google Speech-to-Text

### バックエンド
- **Runtime**: Node.js + Express / Next.js API Routes
- **Database**: PostgreSQL (Supabase)
- **Cache**: Redis
- **Storage**: Cloudflare R2 / AWS S3
- **AI Integration**: 
  - OpenAI GPT-4 (デザイン提案・意図解析)
  - Google Speech-to-Text (音声認識)

### インフラ
- **Hosting**: Vercel (Frontend), AWS ECS (Backend)
- **CDN**: Cloudflare
- **Monitoring**: Datadog / Sentry
- **CI/CD**: GitHub Actions
